# LoGym - 사용자 참여형 장소 기반 운동 관리 및 커뮤니티 서비스

## 1. 프로젝트 개요
**LoGym**은 기존 피트니스 앱들이 개인의 데이터 보관에만 집중하는 한계를 극복하고, 파편화된 '헬스장 인프라 정보'를 사용자 위치 기반으로 명확히 제공하는 웹앱(PWA) 프로젝트입니다. 실제 이용객들이 직접 입력하고 검증한 기구 데이터를 제공함으로써 사용자의 정보 탐색 비용을 줄이고 운동 효율을 극대화하며, 소셜 피드를 통해 상호 동기부여 커뮤니티를 구축합니다.

## 2. 기획 배경 및 근거

### 2.1. 작품 배경
- **현상 파악**: 기존 운동 기록 서비스는 개인 기록에 치중되어 있어, 헬스장의 구체적인 기구나 머신 브랜드 유무를 사전에 파악하기 어렵습니다.
- **새로운 가치 창출**: 흩어져 있는 블로그 후기나 사진에 의존하던 기구의 상세 정보를 사용자 위치와 결합하고, 이용객 참여형 데이터 플랫폼을 제공하여 피트니스 인프라 정보의 불투명성을 해소합니다.

### 2.2. 시장 추세
- **소셜 피트니스와 '오운완' 인증**: 단순한 사진 인증에서 나아가 구체적인 운동 수치와 로그를 타인과 공유하며 동기부여를 얻는 문화가 정착되었습니다.
- **'헬스 유목민' 등장과 스마트 트레이닝**: 본인이 선호하는 특정 브랜드의 머신이 구비된 곳을 찾아다니는 사용자가 늘어남에 따라, 사전 인프라 탐색에 대한 수요가 증가하고 있습니다.

### 2.3. 작품 목표 및 실용적 근거
- **정보 비대칭 해소 지원**: 위치, 가격, 인테리어 위주였던 기존 포털의 한계를 벗어나 필수적인 '기구 정보'를 제공해 앱 내 검색 한 번으로 필요 정보를 탐색할 수 있습니다.
- **운동 지속성 강화 기제**: 사용자 간 운동 로그를 공유하는 소셜 커뮤니티 기능을 통해 운동 지속 가능성을 극대화합니다.

## 3. 핵심 기능 (기술 분석 핵심 요약)

1. **웹앱(PWA) 지원**
   - Vite-PWA 플러그인과 Vercel을 연동하여 오프라인 캐싱 및 앱 기반 홈 화면 아이콘 설치(A2HS) 기능을 제공합니다.
2. **소셜 로그인 연동 (OAuth 2.0)**
   - 카카오 및 구글 API를 호출하여 안전하고 간편하게 사용자 개인화 정보를 생성하고 관리합니다.
3. **위치 기반 검색 및 헬스장 등록**
   - Kakao Map SDK를 바탕으로 주위 헬스장 마커를 띄우고, 특정 점포를 내 계정에 '활성 헬스장'으로 등록합니다.
4. **로컬스토리지 기반 실시간 스톱워치**
   - 타임스탬프와 LocalStorage를 활용하여, 새로고침이나 일시적인 창 이탈 시에도 시간이 이어서 기록되는 스톱워치를 제공합니다.
5. **운동 데이터 CRUD & 서버리스 이미지 업로드**
   - 세트당 무게, 반복 횟수를 저장하고 당일의 오운완 사진을 업로드합니다.
   - Cloudinary를 사용하여 별도로 이미지 서버를 구축할 필요 없이 클라이언트 측에서 직접 최적화된 업로드 URL을 DB로 반환하여 통신 속도를 제고합니다.
6. **친구 조회 및 소셜 피드**
   - 유저 검색을 통해 친구 관계를 맺고, 친구의 운동 시간/종목 요약본을 피드 형태로 확인합니다.
   - Supabase 관리자 권한을 차용하여 '친구 한정 공개 옵션' 등의 RLS 규칙을 기반으로 데이터를 필터링합니다.
7. **운동 캘린더 및 대시보드 시각화**
   - React FullCalendar를 응용하여 출석 및 운동일 색상을 구분하여 동기를 자극하고, 날짜 클릭 시 운동 요약 통계를 띄웁니다.
8. **권한 기반 데이터 검증 (점포 관리자/사용자 구분)**
   - **점포 관리자**: 새로운 헬스장 기구를 등록하거나 수정하며, 사용자의 인프라 추가 요청을 승인 또는 반려합니다.
   - **일반 사용자**: 새로운 기구를 발견하여 등록 요청을 보내며, 이때 기구 이름 타이핑 시 DB의 기존 데이터와 실시간 유사/중복 점검 로직을 거칩니다.

## 4. 구현 환경 및 기술 스택

| 구분 | 기술 스택 | 비고 |
| --- | --- | --- |
| **프론트엔드** | React | UI 설계 및 빌드 속도 최적화 |
| **스타일링** | Tailwind CSS | 모바일 우선 반응형 디자인 구현 |
| **백엔드/데이터베이스** | Supabase (PostgreSQL) | 실시간 통신 및 관계망 데이터 무결성 지원 |
| **PWA 환경** | Vite-PWA | 홈 화면 설치 및 사용자 UX 지원 |
| **사용자 인증** | OAuth 2.0 API | 카카오/구글 간편 로그인 연동 |
| **지도 및 위치** | Kakao Map SDK | 위치 기반 장소 시각화 기능 지원 |
| **이미지 서버** | Cloudinary | 이미지 업로드 및 최적화 반환 |
| **캐싱** | LocalStorage | 스톱워치 타임스탬프 등 데이터 보전 |
| **파이프라인 및 배포** | Vercel | PWA 보안 환경 유지 및 CI/CD 구축 |
